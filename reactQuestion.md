# react 解决了什么问题

1.跨平台组件复用问题
使用React写出来的组件，结合React-dom可以显示在浏览器上，
结合React-native可以显示在android、IOS原生应用上，
可以让我们的组件，一次编写，处处运行，帮助我们避免大量的重复劳动。
2.性能问题
避免不必要的渲染
对于每个组件，它在内存里生成一棵虚拟dom树，当组件发生变化的时候，它会生成一颗新的虚拟dom树，然后和老的进行比较，找出有差异的节点，然后只更新发生变化的dom节点，更新的成本降低，性能提高。
3.提高浏览器兼容性
使用SyntheticEvent，包装和规范了原生的浏览器事件，这样各个浏览器的事件行为都能够得到统一
